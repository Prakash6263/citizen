🎯 QUICK START
─────────────────────────────────────────────────────────────────────

Step 1: Place Firebase Credentials
   - Copy: citixen-app-firebase-adminsdk-xxx.json to project root
   - OR set: FIREBASE_SERVICE_ACCOUNT_PATH in .env

Step 2: Start Server
   $ npm install  (firebase-admin already added)
   $ npm run dev

Step 3: Test Login with FCM Token
   $ curl -X POST http://localhost:5000/api/auth/citizen-login \
     -H "Content-Type: application/json" \
     -d '{"identifier":"email@example.com","password":"pass123","fcmToken":"YOUR_TOKEN"}'

Step 4: Send Test Notification
   $ curl -X POST http://localhost:5000/api/notifications/test \
     -H "Content-Type: application/json" \
     -d '{"email":"email@example.com","title":"Test","body":"Working!"}'

Step 5: Check Your Device for Notification ✅
