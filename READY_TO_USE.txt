🚀 QUICK START IN 5 MINUTES
══════════════════════════════════════════════════════════════════════════

1. PLACE FIREBASE CREDENTIALS
   ➜ Copy: citixen-app-firebase-adminsdk-fbsvc-*.json to project root
   ➜ OR set: FIREBASE_SERVICE_ACCOUNT_PATH in .env

2. START THE SERVER
   ➜ npm install  (firebase-admin already in package.json)
   ➜ npm run dev

3. LOOK FOR SUCCESS MESSAGE
   ✓ "✅ Firebase Admin SDK initialized successfully"

4. TEST LOGIN WITH FCM
   ➜ curl -X POST http://localhost:5000/api/auth/citizen-login \
     -H "Content-Type: application/json" \
     -d '{"identifier":"citizen@example.com","password":"pass123","fcmToken":"YOUR_FCM_TOKEN"}'

5. SEND TEST NOTIFICATION
   ➜ curl -X POST http://localhost:5000/api/notifications/test \
     -H "Content-Type: application/json" \
     -d '{"email":"citizen@example.com","title":"Test","body":"Firebase works!"}'

6. CHECK YOUR DEVICE
   ✓ You should see a push notification!
